Character Encoding

It tells the web browser how to translate a sequence of bytes into readable characters, ensuring that text, symbols, and special characters are displayed correctly and consistently across different devices and languages

UTF-8 encoding allows browsers to correctly display text, symbols, and multiple languages like Arabic, Urdu, and English.

Example symbols:

✓ © ® €

Link Element

The link element connects external resources such as CSS files to an HTML document.
