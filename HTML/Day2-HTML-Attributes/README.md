# Day 2 - HTML Attributes

## What I Learned

HTML attributes provide additional information about elements.
They are written inside the opening tag using this syntax:

<element attribute="value"></element>

## Important Attributes Covered

- href → Specifies link destination
- target="_blank" → Opens link in new tab
- src → Specifies image source
- alt → Alternative text (important for accessibility)
- type → Defines input type
- placeholder → Hint text inside input
- required → Makes input mandatory
- checked → Boolean attribute (checkbox selected by default)
- disabled → Boolean attribute (input cannot be used)

## Key Concept

Some attributes require values (href, src, type).
Some attributes are boolean (checked, disabled, required).

Understanding attributes is essential for:
- Linking pages
- Displaying images
- Creating forms
- Improving accessibility
